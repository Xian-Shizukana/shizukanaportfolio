Games

All of the games listed here can be found on xianshizukana.itch.io!

Between Two Worlds - A 3d platformer game made in Godot wherein you have the ability to flip between two worlds. Not only was this my first game that required actual programming, but it was also my first collaboration game, as this was made by me and my classmates. Our group is currently named Haroshirace.


Wondertap! - A joke clicker game I made about Emu Otori and her infamous "Wonderhoy!" Despite it being a joke game, I did implement features not present in my past game, such as saving, the utilization of classes and global variables.


Mogi Kioku - A game heavily inspired by Yume Nikki. Take control of Kajou as you explore what is behind the 4 doors, collecting pills along the way to progress. This game is my introduction to RPGMaker, and my way of learning the ropes of the engine.


The Day That You Left - This is an hour long visual novel about Haruma, a college student whose childhood friend is going away for unknown reasons. This game centers on how Haruma deals with the said loss, as well as the unexpected friendship he'll build along the way. This game is created in Ren'py, a visual novel engine. It uses Python, and it was easy making the game itself. The only hard part was the writing of the story.


Passing Faces - This is a ~10 minute visual novel about a man's daily life in the cafe. It entirely focuses on the people he meets at the cafe, and how he projects his own fears and doubts onto them. This visual novel was made in only 2 days. It was originally made as a showcase or demo for a workshop I spoke on, but I decided to go against it and make it into its own game.

Websites

Sadly, a lot of these websites are no longer accessible, as they were only done in fulfillment of my subjects' requirements, and were not meant to be kept up.

C.R.O.N.O.S (Check-in Record Organizer for Notifications and Overview of Students) - This is my most “advanced” website to date. This website was done in fulfillment of my Programming Languages 2 subject, and our teacher specifically assigned this task to my group. It is an attendance checker, wherein a QR code scanner automatically inputs the student’s ID number into the field, registering the student as present on that day. It also has a built in tracker, where one can search for the students present during that time. This website utilizes Java with the Spring boot framework, Tomcat, and XAMPP.


XAMPP Login and Register Page - This is a login and register page my group did for our Introduction to Web Design subject. The site utilizes XAMPP to connect the site to a database, and logging in leads to the portfolio of my group. This is coded in PHP.


Room People Counter - This is a program we built for the Quarterly Assessment in Introduction to Web Design back in Grade 11. It's a website shown using an Arduino board, and it displays the number of people currently inside the room.

Arduino Projects

These arduino contraptions were done in fulfillment of our Internet of Things subject in Grade 12.

Tic Tac Toe - This is my Tic Tac Toe project I made for the 3rd Quarter Quarterly Assessment. It features two buttons: one for selecting and one for confirming. The LED indicates whose turn it is, and the entire game happens on the serial monitor.


Project Knox - This is the project my team made for the 4th Quarter Quarterly Assessment. It is a security system that detects intruders in range, and automatically takes a picture of them and sends it to the programmed email. I was one of the programmers for this project, and also the one who conceptualized the idea.
